# AI와 소매 수요 예측 - 실습 코드

이 저장소는「AI와 소매 수요 예측」 책의 실습 코드와 예제 노트북을 제공합니다.

본 저장소에서는 다음과 같은 수요 예측 방법들을 Jupyter Notebook을 이용하여 실습할 수 있습니다.

- 수요 패턴 분석
- Rule-based 수요 분류
- 클러스터링 기반 수요 세분화
- 머신러닝 기반 수요 예측
- 딥러닝 기반 수요 예측
- 시계열 임베딩 기반 분석
- 하이브리드 수요 예측 프레임워크
